使用GitHub+Hexo搭建静态网站

想拥有个人网站或者独立博客的伙伴可以参考参考这里～

Question1:

什么是GitHub？

Answer:

GitHub(原名Logical Awesome LLC)是一个使用git进行版本控制的基于网络的托管服务。它主要用于计算机代码。它提供了Git的所有分布式版本控制和源代码管理（SCM）功能，并添加了自己的功能。它为每个项目提供访问控制和几个协作功能，如错误跟踪，功能请求，任务管理和维基。–摘自维基百科

Question2:

你为什么要选GtiHub？

Answer:

它免费+没有广告+稳定。而且用GitHub感觉更像是自己的东西，有一种归属感。

Question3:

好吧，你开心就好:-D。那Hexo是什么东东？

Answer:

Hexo是一个博客框架，使用它写HTML比较快速，它使用Markdown（渲染引擎）解析文章，利用配置的主题生产静态网页。

下面开始搭建步骤：(搭建环境Linux，Windows请参阅:传送门)

1.注册GitHub账号并创建个人仓库

如果没有GitHub账号，则使用邮箱注册GitHub账号,创建完成后点击GitHub中的New repository创建新仓库，仓库命名为：‘用户名’.github.io ,用户名使用GitHub账号名称，比如我的账号名为JokerWo，则仓库名为JokerWo.github.io

2.安装Git（注意：命令后跟’#‘，从‘#’到行尾为注释，以下同）

点击左侧Edit profile
点击左侧SSH and GPG keys
点击New SSH key之后把刚才输出的公钥内容粘贴到Key栏，Title随便填，之后点击Add SSH Key

3.安装Node.js和Hexo

到Nodejs官网下载对应版本的安装包，然后安装即可。
